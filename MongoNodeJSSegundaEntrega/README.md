``` JavaScript
    npm install
```

Luego, con la terminal posicionada en la carpeta, le dan a 

``` Node
    npm start
```

Este comando en el fondo (en el archivo package.json, en Scripts lo puede ver) ejecuta 
``` Javascript
    node index.js
```
